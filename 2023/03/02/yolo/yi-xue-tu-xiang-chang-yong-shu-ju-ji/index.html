医学图像常用数据集

发表于2023-03-02|更新于2023-03-02|deep learning
|阅读量:

医学图像分类

  1. ChestX-ray14：这是一个14个疾病的胸部X光图像分类数据集，包含112,120张图像，用于检测肺部疾病的影像学表现。
  2. MURA：这是一个用于肌骨疾病检测的数据集，包含40,561张正常和异常的肌骨影像。
  3. ISIC Skin Lesion：这是一个用于皮肤病分类的数据集，包含超过23,000张皮肤病图像，包括恶性黑色素瘤和良性肿瘤。
  4. DDSM：这是一个数字化乳腺X光图像数据集，包含2,620张乳腺X光图像，用于乳腺癌检测和诊断。
  5. NIH Chest X-ray：这是一个用于胸部X光图像分类的数据集，包含超过11万张胸部X光图像，涵盖14种不同的疾病。
  6. BraTS：这是一个用于脑部肿瘤分割和分类的数据集，包含多个MRI扫描序列，包括T1、T1-Gd、T2和FLAIR，每个序列包含多个患者的扫描图像。

这些数据集都是用于医学图像分类任务的，涵盖了多种不同的疾病和图像类型。它们可以用于开发和评估医学图像分类算法，也可以用于医学图像诊断和研究。

医学图像分割

  1. ISBI Challenge: 这是一个由国际生物医学图像分割竞赛组织的数据集。这个竞赛提供了多个医学图像分割任务，如肝脏分割、肺部结节分割等。
  2. BraTS: 这是一个用于脑部肿瘤分割和分类的数据集，包含多个MRI扫描序列，包括T1、T1-Gd、T2和FLAIR，每个序列包含多个患者的扫描图像。该数据集涵盖了四个不同等级的肿瘤分割任务。
  3. LIDC-IDRI: 这是一个肺部结节CT扫描的数据集，包含1018个病人的CT图像和结节注释信息，旨在帮助肺癌的早期检测和诊断。
  4. ACDC: 这是一个心脏分割和功能评估的数据集，包含100个MRI图像序列，涵盖了心脏的不同切面和心脏的不同状态。
  5. Lung Image Database Consortium (LIDC) dataset: 这是一个基于放射学家标注的CT图像肺部结节分割数据集，包含1018个病人的1397个结节。
  6. CVC-ColonDB：这是一个用于结肠镜图像分割的数据集，包含612张RGB结肠镜图像，可以用于结肠肿瘤等疾病的诊断和治疗。

这些数据集提供了不同类型和不同部位的医学图像，可以用于开发和评估医学图像分割算法，也可以用于研究医学影像处理和医学诊断技术的应用。

医学图像检测

  1. ChestX-ray8：这是一个检测8种常见的肺部疾病的胸部X光图像数据集，包含108,948张图像。
  2. NIH Chest X-ray：这是一个用于胸部X光图像检测的数据集，包含超过11万张胸部X光图像，涵盖14种不同的疾病。
  3. DDSM：这是一个数字化乳腺X光图像数据集，包含2,620张乳腺X光图像，用于乳腺癌检测和诊断。
  4. MURA：这是一个用于肌骨疾病检测的数据集，包含40,561张正常和异常的肌骨影像。
  5. PASCAL VOC Medical Image Challenge：这是一个医学图像检测竞赛数据集，涵盖了多种医学图像，如脑部MRI图像、胸部X光图像和眼底图像等。
  6. DDSM：这是一个数字化乳腺X光图像数据集，包含2,620张乳腺X光图像，用于乳腺癌检测和诊断。

这些数据集提供了不同类型和不同部位的医学图像，可以用于开发和评估医学图像检测算法，也可以用于研究医学影像处理和医学诊断技术的应用。
