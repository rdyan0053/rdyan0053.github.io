安装配置

安装termux高级终端以及MT管理器

首先需要安装好ZeroTermux应用，这是酷安『一只小doge』正在开发的新版魔改termux，其次需要安装MT管理器，这是一个文件管理器，可以用来管理手机文件，方便我们操作。

恢复容器操作，下载青龙面板容器恢复包【教程以青龙v2.10.13版为例】

  1. 把你下载好的青龙面板恢复包(后缀名为.tar.gz)放在手机内部存储目录中的xinhao/data路径下
  2. 打开Termux，从左侧向右滑动，点击备份/恢复选项，选择恢复包
  3. 给容器命名为：ql（自己随便命名一个即可），恢复完成后退出Termux重新打开
  4. 再次从左侧向右滑动，点击容器切换选项，点击自己命名的ql容器

启动青龙面板

  1. 先手动启动青龙面板测试一下是否能正常运行

手机打开浏览器访问127.0.0.1:5700，如果出现青龙面板的登录界面，说明青龙面板已经启动成功了。如果电脑和手机在同一局域网下，可以使用电脑去访问手机的ip地址来使用青龙面版（注：不能使用学校的校园网，即使电脑和手机在同一个校园网也无法访问，可以使用手机开热点）
具体流程：

  1. 手机打开Termux，输入ip addr或者ifconifg查看ip地址（要退出alpine系统）
  2. 电脑打开浏览器，输入手机的ip地址加端口号5700

使用电脑连接手机termux，操纵容器：

  1. 进入手机的Termux，输入exit退出alpine系统
  2. 安装openssh并打开，输入pkg install openssh，然后输入sshd打开ssh服务（pkg无法安装的解决办法）
  3. ⼿机上termux查看ip，输⼊命令 ifconfig
  4. ⼿机上termux查看⽤户名，输⼊命令 whoami(例如我的⽤户名是 u0_a469)
  5. ⼿机上termux设置密码，输⼊命令 passwd
  6. 电脑端xshell设置登陆，但是需要注意的是⼿机上termux开启的sshd服务⽤的是8022端⼝，⽽不是常⽤的22端⼝

注：

  1. termux设置⾃动开启ssh
    每次启动termux时都需要⼿动输⼊sshd才能使⽤sshd连接，太⿇烦了，执⾏下⾯的命令，让termux⾃动启动
    echo "sshd" >> ~/.bashrc
  2. 一定要仔细检查ip地址，端口号，⽤户名，密码是否输对了，否则就会连接失败
