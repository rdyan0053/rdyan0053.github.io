卷积参数计算

输入特征图尺寸：(h,w,c)

卷积核大小k，步长s，数量n

卷积参数公式：c * n * k * k + n

计算过程如下：

  1. 只有卷积核有参数，对于一个3*3的卷积核，其参数是 3*3，公式为：k * k
  2. n个卷积核，公式为：n * k * k
  3. 输入特征图有c个通道，每个通道与每个卷积核运算，所以公式是：c * n * k * k
  4. 每个卷积核有一个bias偏执量，n个卷积核就有n个bias，最终公式为：c * n * k * k + n
